CSV Format Requirements

Your CSV file should contain the following columns:

  • Symbol - Asset name (e.g., GBPJPY)
  • Type - Buy or Sell
  • Open Date - Entry date/time
  • Open - Entry price
  • Closed Date - Exit date/time (optional)
  • Closed - Exit price (optional)
  • TP - Take Profit level (optional)
  • SL - Stop Loss level (optional)
  • Lots - Position size
  • Commission - Trading fees (optional)
  • Profit - P&L amount (optional)
